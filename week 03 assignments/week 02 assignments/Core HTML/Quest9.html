Most Common Formatting Tags :

  1. "<b>"
    • The <b> tag in HTML is used to specify the bold text without any extra importance.
    • The text is written within this tag display in bold size.
    • It is a container tag that contains an opening tag, content & closing tag.
  2. "<i>"
    • The <i> tag in HTML is used to display the content in italic style.
    • This tag is generally used to display the technical term, phrase, the important word in a different language.
    • The tags like "b" and "i" now define semantics rather than typographic appearance. So to display text in italic type, users can use the CSS font-style property
  3. "<u>"
    • The <u> tag in HTML stands for underline, and itâ€™s used to underline the text enclosed within the <u> tag.
    • This tag is generally used to underline misspelled words.
    • This tag is depreciated from HTML 4.1 and redefined in HTML 5 using CSS text-decoration property instead.
  4. "<strong>"
    • The <strong> tag in HTML is the parsed tag and used to show the importance of the text.
    • Make that text bold.
    • This tag also supports the global attributes and event attributes in HTML.
